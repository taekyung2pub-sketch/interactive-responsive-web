Convention

State Class — is- / u-

is-active 현재 페이지 nav 링크
is-open LNB, overlay 열린 상태
is-disabled 비활성 링크/버튼
is-visible scroll reveal 완료
is-scrolled nav 스크롤 배경
u-fade scroll reveal 초기 상태

.tag-lang HTML CSS JS TS
.tag-fw React Vue Next Nuxt
