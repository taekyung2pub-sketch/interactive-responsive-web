Overview

직접 설계한 React Component System 위에
실제 서비스 수준의 모바일 쇼핑몰을 구현했습니다.
7개 페이지, 컴포넌트 조립형 구조,
props 설계부터 Portal·상태 관리까지
시스템 레벨의 결정을 직접 담당했습니다.

Focus

  • 컴포넌트 시스템 기반 페이지 조립
  • props 설계 — variant, controlled, Portal
  • 오버레이 UX — BottomSheet 드래그, 스크롤 잠금
  • Swiper 드래그 중 링크 클릭 방지
  • Mock Data 구조 분리 관리

Status

Done

Home — 히어로 배너 Swiper, 카테고리 탭, Best·New·Sale 상품 섹션, 프로모션 배너, 매거진 피드

CommunityDetail — 게시글 본문, 좋아요, 댓글 CRUD(내 댓글 삭제), Textarea 입력
