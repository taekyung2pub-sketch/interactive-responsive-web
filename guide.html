Component Guide

TextField

사용자 입력을 처리하기 위한 기본 입력 컴포넌트입니다. 로그인, 검색, 폼 작성 등 다양한 입력 흐름에서 공통적으로 사용됩니다.

1. Component Role

TextField는 사용자의 텍스트 입력을 담당하는 기본 Form 컴포넌트입니다. 단일 입력 흐름을 기준으로 설계되었으며, 상태 표현과 입력 인터랙션에 집중합니다.

  • 로그인 / 회원가입 입력 UI
  • 검색 및 필터 입력 영역
  • 기본 Form 입력 흐름

2. Composition

  1. FormField와 조합하여 사용
  2. slot 구조 기반으로 확장
  3. 상태 표현은 내부에서 처리
  4. validation은 상위 컴포넌트에서 관리

3. Notes

  • 상태 관리 책임

    입력 상태(focus, disabled, readonly)는 TextField 내부에서 처리합니다.

  • 유효성 관리

    에러 메시지 및 validation 상태는 FormField에서 관리합니다.

  • 확장 기준

    아이콘, 버튼 등은 slot 구조를 기준으로 확장합니다.

    • 검색 아이콘 추가
    • 비밀번호 토글 버튼
    • 상태 아이콘 노출
  • 협업 기준

    스타일 수정 시 spacing / typography token 사용을 권장합니다.
